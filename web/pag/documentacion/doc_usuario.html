Documentación de Usuario

Bienvenido a Squirrel Games, un simulador de gestión inspirado en los famosos juegos tipo Squid Game. Este software te permitirá gestionar una edición completa de los juegos, controlando participantes, pruebas, y el personal de la organización.

¿Qué necesitas para jugar?

  • Java 17 o superior instalado en tu equipo.
  • Maven correctamente configurado.
  • Un IDE compatible (Eclipse, IntelliJ, NetBeans o similar).

Cómo ejecutar el juego

  1. Clona o descarga el proyecto desde el repositorio.
  2. Importa el proyecto como proyecto Maven.
  3. Ejecuta la clase Main.java.
  4. Sigue las instrucciones en pantalla para crear una edición de los juegos, registrar participantes, asignar roles, crear pruebas y simularlas.

Características principales para el usuario

  • Creación dinámica de participantes.
  • Definición de pruebas personalizadas.
  • Simulación automática de pruebas con eliminaciones aleatorias.
  • Gestión de jerarquía del personal (Pink Guards, Soldiers, Managers).
  • Control de excepciones para reglas clave.

¡Disfruta organizando tu propia edición de Squirrel Games y recuerda: el juego solo acaba cuando queda un único vencedor!
